# Travel Media Brief Jekyll Site

A Jekyll conversion of the static Travel Media Brief site with blog support and topic/domain pages.

## Includes

- Jekyll layouts and includes
- Blog index and sample posts
- Topic collection in `_topics/`
- Working `/topics/` page
- Substack call-to-action
- Preserved theme toggle and original visual style

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
